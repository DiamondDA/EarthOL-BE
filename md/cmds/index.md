## 这些命令全是操作员命令，普通玩家无法使用
| 命令作用| 命令名称 |
| ----- | ----------- |
| 禁止TNT爆炸 |      gamerule tntExplodes false       | 
| 禁止生物破坏 | gamerule mobGriefing false   |      
| 禁止火焰蔓延 | gamerule tntExplodes false   |         
| 死亡不掉落 | gamerule doMobLoot true   |         
| 踢出玩家 | kick 玩家ID |  
| 给予玩家操作原权限 | op 玩家ID   |         
| 显示坐标 | gamerule showcoordinates true  |
